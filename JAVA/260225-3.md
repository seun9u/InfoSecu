## 실습.[LAB] 리눅스에서 Java 설치하기

https://openjdk.org/

### 1. 패키지 검색 및 설치
```bash
root@ubuntu:~# apt search openjdk
root@ubuntu:~# apt -y install openjdk-11-jdk vim
```

---

### 2. vi (vim) 편집기 기초 사용법
* **입력 모드:** 글자를 입력하는 모드 (명령 모드에서 **`i`** 입력)
* **ex 모드:** 저장 및 종료 명령 수행
    - **`:w`** : 저장
    - **`:q`** : 종료
    - **`:wq`** : 저장 후 종료
    - **`:q!`** : 저장하지 않고 종료

**연습 예시:**
`vim test.txt` > `i` > `Hello` 입력 > `ESC` > `:wq`
```bash
root@ubuntu:~# vim test.txt
Hello
```

```bash
root@ubuntu:~# javac -version
javac 11.0.30
```

---

### 4. 자바로 소스코드와 실행파일 생성 순서


1. **소스코드 작성:** `vim Hello.java`
2. **컴파일:** `javac Hello.java` 실행 (클래스 파일 생성)
3. **실행:** `java Hello` 실행

---

### 5. Java 실습 코드 작성 및 실행

```bash
# vim 내부에서 가독성을 위해 설정 가능
:set bg=dark
```

#### [Hello.java]
```java
// Hello.java

public class Hello {

        public static void main(String[] args) {
                System.out.printf("Hello world!\n");
        }

}
```

#### [컴파일 및 실행]
```bash
root@ubuntu:~# javac Hello.java
root@ubuntu:~# java Hello
Hello world!
```
