# 🛠️ C++ 기초 실습 요약 정리 (Dev C++)

## 1. 개발 환경 및 기본 출력
Dev C++에서는 프로젝트 단위 또는 개별 파일(`.cpp`) 단위로 개발이 가능합니다. C++의 표준 출력은 `std::cout`을 사용합니다.

```cpp
// hello.cpp
#include <iostream>

int main() {
    // std::cout을 이용한 문자열 출력
    std::cout << "Hello C++\n";
    return 0;
}
```

---

## 2. 제어문 (while, for)
C언어와 동일한 제어 구조를 가집니다. 표준 출력 시 `std::endl` 또는 `'\n'`을 사용하여 줄바꿈을 처리합니다.



### [while문 실습]
```cpp
// while1.cpp (1~5까지 출력)
#include <iostream>

int main() {
    int i = 1; 
    while(i <= 5) {
        std::cout << i << '\n';
        i++;
    }
    return 0;
}
```

### [for문 실습]
```cpp
// for1.cpp (1~5까지 출력)
#include <iostream>

int main() {
    for(int i = 1; i <= 5; i++) {
        std::cout << i << '\n';
    }
    return 0;
}
```

## 3. 네임스페이스 (Namespace)
이름 충복을 방지하기 위한 공간입니다. `using` 지시자를 사용하면 `std::` 접두어를 생략할 수 있습니다.

* **사용 형식:** `네임스페이스명::변수/함수;`
* **주요 지시자:** `using namespace std;`

```cpp
// namespace1.cpp
#include <iostream>
using namespace std; // std 네임스페이스 생략 가능하게 함

int main() {
    cout << "Hello C++" << endl; // using 지시자 덕분에 cout만 사용 가능
    return 0;
}
```

## 4. 클래스 (Class)와 정보 은닉
구조체가 발전된 형태로, 객체를 만들기 위한 **설계도**입니다. 멤버 변수(속성)와 멤버 함수(메서드)로 구성됩니다.



### 접근 지정자 (Access Specifiers)
* **public:** 외부 접근 가능
* **private:** 외부 접근 불가 (클래스의 기본값)
* **protected:** 파생 클래스에서는 public, 외부에서는 private 취급

### [클래스 정의 및 호출 실습]
멤버 함수를 클래스 내부에 정의하거나, 외부에서 `::`(범위 지정 연산자)를 사용하여 정의할 수 있습니다.

```cpp
// class2.cpp
#include <iostream>
using namespace std;

class Test {
public: // 접근 권한 설정
    int x; // 멤버 변수
    int y;
    void sum(int a, int b); // 멤버 함수 선언
};

// 클래스 외부에서 멤버 함수 정의
void Test::sum(int a, int b) {
    cout << "sum = " << a + b << endl;
}

int main() {
    Test test; // 객체 생성
    test.x = 1;
    test.y = 2;
    test.sum(test.x, test.y); // 멤버 함수 호출
    
    return 0;
}
```
