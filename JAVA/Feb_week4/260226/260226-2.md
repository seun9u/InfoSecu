## 실습.[LAB] C언어 소스코드 Java로 포팅(변환) 하기

### 1. 원본 C 코드 (분할 컴파일 구조)
```c
// main.c -> Main > main
#include <stdio.h>
void test_func1();
void test_func2();

int main(){
    test_func1();
    test_func2();
    return 0;
}

// test_func1.c -> TestFunc1.java > testFunc1
#include <stdio.h>
void test_func1(){
    printf("test_func1() 호출!\n");
}

// test_func2.c -> TestFunc2.java > testFunc2
#include <stdio.h>
void test_func2(){
    printf("test_func2() 호출!\n");
}
```

---

### 2. 자바로 변환한 코드 (section03 패키지)



#### [TestFunc1.java]
```java
package section03;

public class TestFunc1 {
    // 접근제어자, 접근 제한자
    // public: 다른 패키지에서 접근을 허용한다.
    // private: 같은 패키지나 다른 패키지에서 접근을 금지한다. 같은 클래스에서만 접근이 가능하다.
    // default (package-private): 다른 패키지에서 접근을 금지하고 같은 패키지에서는 허용한다.
    void testFunc1()
    {
        System.out.print("test_func1() 호출!\n");
    }
}
```

#### [TestFunc2.java]
```java
package section03;

public class TestFunc2 {
    void testFunc2()
    {
        System.out.print("test_func2() 호출!\n");
    }
}
```

#### [Main.java]
```java
package section03;

public class Main{
    public static void main(String[] args) {
        // 객체 생성 후 메서드 호출
        TestFunc1 testfunc1 = new TestFunc1();
        TestFunc2 testfunc2 = new TestFunc2();
        testfunc1.testFunc1(); 
        testfunc2.testFunc2();
    }
}
```

### 3. Java 반복문 실습 예제 (package5 패키지)

#### [Ex05_01.java] - for문 기본
```java
package package5;

public class Ex05_01 {
    public static void main(String[] args) {
        // 1 ~ 5까지 출력
        for(int i=1; i <= 5; i++)
            System.out.println(i);
    }
}
```

#### [Ex05_02.java] - for문과 if문(짝수 출력)
```java
package package5;

public class Ex05_02 {
    public static void main(String[] args) {
        // for 문을 이용해서 1 ~ 10까지 반복할 때 짝수만 출력한다.
        for(int i=1; i <= 10; i++)
        {
            if(i % 2 == 0)
                System.out.println(i);
        }
    }
}
```

#### [Ex05_05.java] - while문
```java
package package5;

public class Ex05_05 {
    public static void main(String[] args) {
        // while 문을 이용해서 1 ~ 5까지 출력한다.
        int i = 1;
        while(i <= 5)
        {
            System.out.println(i++);
        }
    }
}
```

#### [Ex05_06.java] - do~while문
```java
package package5;

public class Ex05_06 {
    public static void main(String[] args) {
        // do ~ while 문을 이용해서 1 ~ 5까지 출력한다.
        int i = 1;
        do 
        {
            System.out.println(i++);
        } while(i <= 5); 
    }
}
```
