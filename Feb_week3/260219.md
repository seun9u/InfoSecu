# ♾️ 무한 루프와 반복문 (while, do~while, for)

## 1. 무한 루프 (Infinite Loop)
* 조건식에 **항상 참(True)**을 만들어서 반복문을 실행하는 것을 말합니다.
* 항상 조건이 참이므로 반복문을 스스로 탈출할 수 없을 때를 말합니다.
* 따라서 실행문 내부에서 **if문**과 **break문**을 이용해서 반복문(루프)을 강제로 탈출해야 합니다.

## 2. while 문
* 반복할 때 사용하는 제어 구조입니다.
* **동작 원리:**
    * 조건식을 비교하고 **참**이면 실행문을 실행합니다.
    * 조건식을 비교하고 **거짓**이면 실행문을 실행하지 않습니다.

### 형식
```c
while(조건식)
{
    실행문;
    // ...
}
```

## 3. do ~ while 문
* 반복할 때 사용하는 제어 구조입니다.
* **동작 원리:**
    * 먼저 **실행문을 한 번 실행**합니다.
    * 그 후 조건식을 비교하여 참이면 계속 반복하고, 거짓이면 종료합니다.

### 형식
```c
do
{
    실행문;
    // ...
} while(조건식);
```

## 4. for 문 (while문과 비교)

while문의 구조를 분석해보면 `for`문의 필요성을 알 수 있습니다.

### while문의 4단계 구조
```c
// 순서: 1.초깃값 > 2.조건식 > 3.실행문 > 4.증감식

int i = 1;              // [1] 초깃값

while(i <= 5)           // [2] 조건식
{
    printf("%d\n", i);  // [3] 실행문
    i++;                // [4] 증감식
}
```
