# 🔤 문자 배열과 문자열 (String)

C언어에는 기본적으로 문자열(String)을 위한 전용 자료형이 없기 때문에, **문자(`char`)들의 배열**을 이용하여 문자열을 표현합니다.

## 1. 문자열의 핵심: NUL 문자 (`\0`)
* 문자들의 모임인 문자열이 끝났음을 컴퓨터에게 알려주기 위해, 문자열의 맨 마지막에는 반드시 **NUL 문자(`\0`)**가 들어가야 합니다.
* 따라서 `N`글자의 문자열을 저장하려면 최소 **`N + 1`** 크기의 배열이 필요합니다.

---

## 2. 실습: 문자를 배열로 저장하고 출력하기 (`array5.c`)

```c
// array5.c
// 문자를 배열로 저장하기
#include <stdio.h>

int main()
{
    char ch[7];  // 전체크기 : sizeof(char) x 7 = 7byte가 메모리에 할당된다.

    // 문자열을 선언과 동시에 초기화하는 다양한 방법들 (아래는 모두 동일한 결과)
    // char ch[]  = "ABCDEF";
    // char ch[7] = "ABCDEF";
    // char ch[]  = { 'A', 'B', 'C', 'D', 'E', 'F', '\0' };
    // char ch[7] = { 'A', 'B', 'C', 'D', 'E', 'F', '\0' };

    // 문자열을 초기화하지 않고 하나씩 넣을 때는 아래처럼 넣을 수 있다.
    ch[0] = 'A';
    ch[1] = 'B';
    ch[2] = 'C';
    ch[3] = 'D';
    ch[4] = 'E';
    ch[5] = 'F';
    ch[6] = '\0';  // NUL 문자 (문자의 모임인 문자열의 끝은 NUL문자가 있어야 한다.)

    /*
     * ch의 배열 메모리 구조
     *
     * ch
     * +---+---+---+---+---+---+----+
     * | A | B | C | D | E | F | \0 |
     * +---+---+---+---+---+---+----+
     * |   |   |   |   |   |   |
     * |   |   |   |   |   |   +-- ch[6]
     * |   |   |   |   |   +-- ch[5]
     * |   |   |   |   +-- ch[4]
     * |   |   |   +-- ch[3]
     * |   |   +-- ch[2]
     * |   +-- ch[1]
     * +-- ch[0]
     */

     // 1. %c를 사용하여 문자 단위로 출력
     printf("%c %c %c \n", ch[0], ch[1], ch[2]);  // 출력: A B C 

     // 2. %s를 사용하여 문자열 전체 출력
     // int printf(const char *restrict format, ...);
     // %s는 시작 주소를 받아서 '\0'을 만날 때까지 연속해서 출력한다.
     
     printf("%s\n", ch);      // 출력: ABCDEF (배열의 이름은 시작 주소)
     printf("%s\n", &ch[0]);  // 출력: ABCDEF (0번 방의 주소 역시 시작 주소)

    return 0;
}
```
