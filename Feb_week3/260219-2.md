# 🛑 반복문 제어: break와 continue 완벽 정리

반복문(loop)의 흐름을 강제로 제어할 때 사용하는 키워드입니다. 
**주의사항:** 두 제어문 모두 반복문 밖에서 단독으로 사용할 수 없으며, 조건 없이 사용하면 루프가 비정상적으로 동작하므로 반드시 `if`문과 함께 사용해야 합니다.

---

## 1. break 문



* **역할:** 만나는 즉시 **반복문 전체를 완전히 빠져나갑니다.**
* `switch ~ case`문의 `break`와는 역할이 다르며, 루프 안에서 사용될 때만 루프를 탈출합니다.

### 1) 잘못된 사용 예시 (조건 없는 break)
루프가 시작되자마자 `break`를 만나 1번만 실행되고 종료됩니다.
```c
// break1_error.c
#include <stdio.h>

int main()
{
    int i = 1;
    for(; i <= 5;)
    {
        printf("i = %d\n", i);
        break;  // 조건 없이 만나서 바로 탈출해버림
        i++;    // 이 코드는 영원히 실행되지 않음
    }
    return 0;
}
// 출력: i = 1
```

### 2) 올바른 사용 예시 (if문과 결합)
조건을 만족할 때만 `break`가 실행되어 루프를 탈출합니다.
```c
// break1_fixed.c
#include <stdio.h>

int main()
{
    int i = 1;
    for(; i <= 5;)
    {
        printf("i = %d\n", i);
        if(i == 5)     // i가 5가 되면
            break;     // 반복문을 탈출!
        i++;
    }
    return 0;
}
// 출력: i = 1, i = 2, i = 3, i = 4, i = 5
```

---

## 2. continue 문



* **역할:** 루프를 완전히 빠져나가는 것이 아니라, **이번 회차의 남은 코드만 건너뛰고 다음 반복으로 넘어갑니다.**
* **루프 종류에 따른 동작 차이 (매우 중요):**
    * **`while`문:** `continue`를 만나면 **조건식**으로 바로 올라갑니다.
    * **`for`문:** `continue`를 만나면 **증감식**으로 바로 올라갑니다.

### 1) 무한 루프 발생 주의 (`continue2.c`)
증감식이 `for`문 헤더가 아닌 본문에 있을 때 `continue`를 만나면 증감식을 건너뛰게 되어 무한 루프에 빠집니다.
```c
// continue2.c
#include <stdio.h>
#include <unistd.h>  // sleep() 함수용

int main()
{
    int i = 1;
    for(; i <= 5;)
    {
        printf("i = %d\n", i);
        sleep(1);  // 1초 대기
        
        if(i == 3)
            continue; // i가 3일 때 위로 올라가버림 (i++ 건너뜀)
            
        i++; // i가 3일 때는 이 코드가 실행되지 않아 영원히 3에 머뭄
    }
    return 0;
}
// 출력: 1, 2, 3, 3, 3, 3, 3 ... (무한 루프)
```

### 2) 증감식 위치 수정 (`continue3.c`)
증감식을 `for`문 헤더에 넣으면 `continue`를 만나더라도 증감식을 거치게 되므로 무한 루프가 해결됩니다.
```c
// continue3.c
#include <stdio.h>

int main()
{
    int i = 1;
    // continue를 만나면 바로 i++ 로 점프함
    for(; i <= 5; i++) 
    {
        printf("i = %d\n", i);
        if(i == 3)
            continue;
    }
    return 0;
}
// 출력: 1, 2, 3, 4, 5
```

### 3) 특정 조건 건너뛰기 완벽 예제 (`continue4.c`)
`i == 3`일 때 출력을 생략하고 다음으로 넘어갑니다.
```c
// continue4.c
#include <stdio.h>

int main()
{
    int i = 1;
    for(; i <= 5; i++)
    {
        if(i == 3)        // i가 3이면
            continue;     // 아래 출력을 무시하고 i++ 로 올라감
            
        printf("i = %d\n", i);
    }
    return 0;
}
// 출력: i = 1, i = 2, i = 4, i = 5  (3이 건너뛰어짐)
```

### 4) 잘못된 사용 예시 (조건 없는 continue)
조건 없이 `continue`를 남발하면 그 아래 코드는 아예 실행되지 않는 데드 코드(Dead Code)가 됩니다.
```c
// continue5.c
#include <stdio.h>

int main()
{
    int i = 5;
    for(; i <= 5; i++)
    {
        continue;  // 만나자마자 바로 i++로 올라감
        printf("i = %d\n", i);  // 절대 실행되지 않음
    }
    return 0;
}
// 출력: (아무것도 출력되지 않음)
```
