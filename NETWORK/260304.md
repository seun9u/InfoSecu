# 🌐 네트워크 기초 핵심 요약 정리

## 1. 네트워크 분류 (LAN vs WAN)
네트워크는 관리 주체와 지리적 범위에 따라 크게 두 가지로 나뉩니다.

* **LAN (Local Area Network)**: 지역 네트워크로, 회사의 관리자(네트워크/서버팀)가 직접 관리합니다.
* **WAN (Wide Area Network)**: 광역 네트워크로, ISP 업체가 관리합니다.
    * **ISP (Internet Service Provider)**: KT, LGU+, SK Broadband 등과 같은 통신 사업자를 의미합니다.
* **스타링크 (Starlink)**: 일론 머스크가 운영하는 위성 통신 기술로, 국내에도 도입되었습니다.

---

## 2. 하드웨어 주소와 통신 매체
PC, 서버, 네트워크 프린터 등이 통신하기 위해서는 물리적인 연결 장치가 필요합니다.



* **LAN 카드 (NIC)**: 통신을 가능하게 하는 카드로, 흔히 랜카드라고 부릅니다.
* **MAC 주소 (Media Access Control address)**: 랜카드 안에 존재하는 고유한 물리 주소입니다.
    * **주의**: 같은 LAN상에 동일한 MAC 주소가 존재하면 **충돌**이 발생합니다.

---

## 3. 프로토콜 (Protocol)
통신할 때 사용하는 약속 또는 규약을 의미합니다. 서로 다른 프로토콜을 사용하면 통신이 불가능합니다.

* **예시**: 네이버 서버가 **HTTP**로 통신 대기 중일 때, 클라이언트가 **SSH** 프로토콜로 접근하면 통신이 되지 않습니다.

---

## 4. 네트워크 표준 (IEEE 802)
전 세계적인 통신 표준은 **IEEE(아이트리플이)**의 LMSC 위원회에 의해 유지보수됩니다.



* **IEEE 802.3**: 이더넷(Ethernet)과 관련된 표준으로, 물리 계층과 데이터 링크 계층의 매체 접근 제어(MAC)를 정의합니다.
* **이더넷 (Ethernet)**: 가장 널리 사용되는 LAN 기술 표준입니다.
