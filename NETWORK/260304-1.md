# 🌐 패킷 트레이서를 이용한 네트워크 구축 기초

## 1. 이더넷 인터페이스 속도 구분
스위치와 라우터에는 다양한 속도의 포트가 존재하며, 전송 속도에 따라 명칭이 달라집니다.

* **Ethernet**: 10Mbps
* **FastEthernet (Fa)**: 100Mbps
* **GigabitEthernet (Gi)**: 1Gbps (1,000Mbps)

---

## 2. IP 주소와 서브넷 마스크 (IPv4)
IP 주소 뒤의 `/24`와 같은 표기는 네트워크 범위를 결정하는 **Bit 수**를 의미합니다.



* **네트워크 주소 예시**: `192.168.1.0/24`
* **서브넷 마스크 계산**:
    - `24bit` = `11111111.11111111.11111111.00000000` (2진수)
    - **10진수 표기**: `255.255.255.0`

---

## 3. 네트워크 통신의 4대 요소
윈도우 실행창에서 `ncpa.cpl` 명령어를 입력하면 네트워크 어댑터 설정에서 아래 항목들을 확인할 수 있습니다.

| 항목 | 중요도 | 역할 |
| :--- | :--- | :--- |
| **IP 주소** | **필수** | 네트워크상에서 장치를 식별하는 고유 주소 |
| **서브넷 마스크** | **필수** | 네트워크 부분과 호스트 부분을 구분하는 기준 |
| **기본 게이트웨이** | 옵션 | 외부 네트워크(WAN)로 나가기 위한 통로 (공유기/라우터 주소) |
| **DNS 주소** | 옵션 | 도메인(naver.com 등)을 IP 주소로 변환해주는 서버 주소 |

> **참고**: DNS 서버가 설정되어 있지 않으면 `www.google.com` 같은 이름은 사용할 수 없으며, 오직 `142.250.196.110`과 같은 **IP 주소로만 접속**이 가능합니다.

---

### 🛠️ 패킷 트레이서 실습 팁
* **PC 설정**: `Desktop` 탭 -> `IP Configuration`에서 위 4가지 요소를 입력합니다.
* **연결 확인**: `Command Prompt`에서 `ping [상대방 IP]`를 입력하여 통신 여부를 확인합니다.
