# 📦 C언어 함수 (Function) 완벽 정리

## 1. 함수(Function)란?
* **정의:** 여러 실행문(명령어)들을 모아놓은 하나의 실행문들의 집합입니다.
* **목적:** 코드의 **재사용성**을 높이기 위해 사용합니다.
* **동작 원리:**
    1. 함수를 호출하면 제어권이 함수가 정의된 곳으로 점프(Jump)합니다.
    2. 함수 내의 실행문들을 차례로 실행합니다.
    3. 실행이 끝나면 함수를 호출했던 다음 위치로 복귀(Return)합니다.

---

## 2. 함수의 종류
1. **라이브러리 함수 (Library Function)**
    * C언어 설치 시 기본적으로 제공되는 함수들
    * 예: `printf()`, `scanf()` 등
2. **사용자 정의 함수 (User-Defined Function)**
    * 프로그래머가 필요에 의해 직접 만들어 사용하는 함수들
    * 예: `myfunction()`, `sum()` 등
    * *대부분의 프로그램은 사용자 정의 함수를 통해 작성됩니다.*

---

## 3. 함수의 주요 용어
* **함수 정의 (Definition):** 실제 함수의 기능을 만드는 것 (코드 구현).
    * 반드시 함수 밖에서 정의해야 하며, 함수 내부에서 정의하면 에러가 발생합니다.
* **함수 선언 (Declaration):** 컴파일러에게 "이런 함수가 있다"고 미리 알려주는 것.
    * `main()` 함수 위에 작성하며, 함수의 머리 부분(Prototype)만 적습니다.
* **함수 호출 (Call):** 만들어진 함수를 사용하는 것.
    * 반드시 다른 함수(예: main) 내부에서 호출해야 합니다.

---

## 4. 입력값과 출력값

### 1) 입력값 (Input)
호출하는 곳에서 함수로 전달하는 값입니다.
* **인수 (Argument):** 함수를 호출할 때 던져주는 실제 값.
    * 예: `f(5);` 에서 `5`
* **매개변수 (Parameter):** 함수가 호출될 때 값을 받아주는 변수.
    * 예: `int f(int x)` 에서 `x`

### 2) 출력값 (Output)
함수가 작업을 마치고 호출한 곳으로 돌려주는 값입니다.
* **리턴값 (Return Value):** 계산된 결과를 반환합니다.
* **참고:** C언어에서 리턴값은 CPU의 **EAX 레지스터**에 저장되어 전달됩니다.

---

## 5. 함수의 구조와 정의
```c
리턴자료형 함수명([매개변수])   // [함수의 머리]
{                           // [함수의 몸체]
    실행할 명령어;
    ...
    [return 값;]            // 생략 가능 (void일 경우)
}
```

### 리턴 자료형의 규칙
* 정수 반환: `int`
* 실수 반환: `float`, `double`
* 주소 반환: `포인터`
* **반환값이 없음**: `void`

### 함수의 원형 (Prototype)
함수의 머리 부분만 사용하여 컴파일러에게 정보를 제공합니다.
* 예: `int printf(const char *format, ...);`

## 6. 예제 코드: 함수의 정의와 선언 위치

### Case 1: 함수 정의를 main() 위에 작성 (`function1.c`)
```c
#include <stdio.h>

// 함수 정의 (만들기)
void myprint()
{
    printf("myprint() 함수 호출!!!\n");
}

int main()
{
    myprint();  // 함수 호출
    return 0;
}
```

### Case 2: 함수 선언을 활용 (`function2.c`)
```c
#include <stdio.h>

// 함수 선언 (알려주기)
void myprint();

int main()
{
    myprint();  // 함수 호출
    return 0;
}

// 함수 정의 (구현)
void myprint()
{
    printf("myprint() 함수 호출!!!\n");
}
```

### Case 3: 매개변수와 리턴값이 있는 함수 (`function3.c`)
```c
#include <stdio.h>

int plus(int x, int y); // 선언

int main()
{
    int sum;
    
    sum = plus(1, 2); // 호출 (인수 1, 2 전달)
    printf("sum = %d\n", sum); // 출력: 3
    return 0;
}

int plus(int x, int y)  // 정의 (매개변수 x, y)
{
    return x + y;       // 값 반환
}
```

## 7. [LAB] 함수의 4가지 형태 실습
> **목표:** `x`의 값 **8**을 출력하는 함수를 4가지 형태로 구현하시오.

### 형태 1: 인수가 없고, 리턴값도 없는 함수
* 받는 것도 없고, 주는 것도 없으므로 함수 내부에서 직접 출력해야 합니다.
```c
// ex_func1.c
#include <stdio.h>

void function1(void);

int main()
{   
    function1();
    return 0;
}

void function1(void)
{
    int x = 8;
    printf("x = %d\n", x);
}
// 출력 결과: x = 8
```

### 형태 2: 인수가 있고, 리턴값이 있는 함수
* 값을 받아서 처리 후, 결과를 다시 `main`으로 돌려줍니다.
```c
// ex_func2.c
#include <stdio.h>

int fuction2(int x);

int main()
{
    int x;
    x = fuction2(5);
    printf("x = %d\n", x);
    return 0;
}

int fuction2(int x)
{
    x = x + 3; // 5 + 3 = 8
    return x;
}
// 출력 결과: x = 8
```

### 형태 3: 인수가 있고, 리턴값이 없는 함수
* 값을 받지만 돌려주지 않으므로, 함수 내부에서 출력합니다.
```c
// ex_func3.c
#include <stdio.h>

void fuction3(int x);

int main()
{
    fuction3(5);
    return 0;
}

void fuction3(int x)
{
    x = x + 3;
    printf("x = %d\n", x);
}
// 출력 결과: x = 8
```

### 형태 4: 인수가 없고, 리턴값이 있는 함수
* `void`는 주로 생략합니다. 내부에서 값을 만들어 `main`으로 돌려줍니다.
```c
// ex_func4.c
#include <stdio.h>

int fuction4(void);

int main()
{
    int x;
    x = fuction4();
    printf("x = %d\n", x);
    return 0;
}

int fuction4(void)
{
    int x = 8;
    return x;
}
// 출력 결과: x = 8
```
