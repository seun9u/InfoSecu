# 📍 구조체 포인터와 구조체 함수

리눅스 시스템 파일인 `/etc/passwd`의 데이터를 다룰 때도 구조체는 매우 중요하게 사용됩니다. 구조체의 주소를 다루는 포인터와 이를 함수로 전달하는 방법을 분석합니다.

```bash
$ ls -l /etc/passwd
-rw-r--r-- 1 root root 2922  2월 12 10:29 /etc/passwd

$ cat /etc/passwd
root:x:0:0:root:/root:/bin/bash
daemon:x:1:1:daemon:/usr/sbin:/usr/sbin/nologin
bin:x:2:2:bin:/bin:/usr/sbin/nologin
  :
  :(생략)
```

---

## 1. 구조체 포인터 (Structure Pointer)

구조체의 주소를 저장하는 포인터 변수입니다.

### 1) 구조체 포인터 선언
* **일반적인 선언:**
  ```c
  struct jumsu {
      int kor;
      int eng;
  };
  struct jumsu j;
  struct jumsu *p; 
  p = &j;
  ```
* **typedef를 이용한 선언:**
  ```c
  typedef struct {
      int kor;
      int eng;
  } JUMSU;

  JUMSU j;
  JUMSU *p; 
  p = &j;
  ```

### 2) 구조체 멤버 접근 및 연산자
* **`.` (Direct Reference):** 직접 참조, 일반 변수일 때 멤버에 접근하기 위해 사용합니다.
* **`->` (Indirect Reference):** 간접 참조, 포인터 변수일 때 멤버에 접근하기 위해 사용합니다.

```c
struct member m;
struct member *p;
p = &m;

m.name  : m 변수의 name 멤버에 접근 (직접 참조)
m.phone : m 변수의 phone 멤버에 접근 (직접 참조)
m.age   : m 변수의 age 멤버에 접근 (직접 참조)

p->name  : m 변수의 name 멤버에 접근 (간접 참조)
p->phone : m 변수의 phone 멤버에 접근 (간접 참조)
p->age   : m 변수의 age 멤버에 접근 (간접 참조)
```



---

## 2. 실습 1: 구조체 포인터 활용 (`struct7.c`)

### 소스 코드
```c
// struct7.c
#include <stdio.h>

typedef struct
{
    char name[20];
    int age;
} MEMBER;

int main()
{
    MEMBER m = { "Hgd", 20 };
    MEMBER *p;  // 포인터 변수이므로 선언 시 초기화 리스트 { } 직접 사용 불가

    p = &m;

    printf("name = %s, age = %d\n", m.name, m.age );
    printf("name = %s, age = %d\n", (*p).name, (*p).age ); // *p로 접근 후 . 참조
    printf("name = %s, age = %d\n", p->name, p->age );     // 화살표 연산자로 간접 참조

    return 0;
}
```

### 실행 결과
```bash
$gcc -Wall -g -o struct7 struct7.c$ ./struct7
name = Hgd, age = 20
name = Hgd, age = 20
name = Hgd, age = 20
```

## 3. 구조체 함수 (Structure and Function)

구조체를 함수의 인자로 넘기거나 함수에서 구조체를 리턴할 수 있습니다.

### 1) 함수 호출 시 구조체를 인수로 넘기는 방법
```c
struct member m;
print_member(m); // 구조체 변수 자체를 전달

void print_member(struct member m)
{
    // 함수 내부에서 멤버 출력
}
```

### 2) 실습 2: 구조체 전달 프로그램 (`struct8.c`)


### 소스 코드
```c
// struct8.c
#include <stdio.h>

struct member
{
    char name[20];
    int age;
};

void print_member(struct member m);

int main()
{
    struct member m = { "Hgd", 20 };

    print_member(m); // Call by Value 형식으로 복사본 전달

    return 0;
}

void print_member(struct member m)
{
    printf("name = %s, age = %d\n", m.name, m.age );
}
```

### 실행 결과
```bash
$gcc -Wall -g -o struct8 struct8.c$ ./struct8
name = Hgd, age = 20
```
