# 🚀 포인터 배열과 명령행 인자 (argc, argv) 분석

## 1. 포인터 배열의 초기화와 구조 (`array14.c`)

포인터 배열은 문자열 리터럴이 저장된 메모리의 주소값을 요소로 가지는 배열입니다.

### 실습 소스 코드
```c
// array14.c
#include <stdio.h>

int main()
{
    // 포인터 배열의 초기화
    char array[] = "Hello C!";  // { 'H', 'e', 'l', 'l', 'o, ' ', 'C', '!', '\0' };
    char *parray[] = { "Linux", "Unix", "Windows", "Mac", NULL };
    
    /*
       parray 메모리 구조 도식
         [0]   [1]   [2]   [3]   [4]
       +-----+-----+-----+-----+-----+
       | 100 | 200 | 300 | 400 | 0x0 | (0x0000000000000000)
       +-----+-----+-----+-----+-----+
       |       |     |       |         |
       |       +--+  |       |         |
       |          |  +--+    |         |
       |          |     |    +-----------+
       |          |     +---+            |
       v          v         v            v
       +-------+  +------+  +---------+  +-----+
       |Linux\0|  |Unix\0|  |Windows\0|  |Mac\0|
       +-------+  +------+  +---------+  +-----+
       100        200       300          400
     */

    printf("array = %s\n", array);
    printf("array = %s\n", &array[0]);
    printf("parray[0] = %s\n", parray[0]);  // Linux
    printf("parray[1] = %s\n", parray[1]);  // Unix
    printf("parray[2] = %s\n", parray[2]);  // Windows
    printf("parray[3] = %s\n", parray[3]);  // Mac

    // for문을 이용한 출력
    putchar('\n');  // printf("\n"); 또는 putchar('\n'); 동일하게 줄바꿈 실행

    // 배열: 문자 출력
    printf("array = ");
    for(int i=0; i < sizeof(array); i++)
        printf("%c", array[i]);

    putchar('\n');

    // 포인터 배열: 문자열 출력 (마지막 NULL을 활용한 조건식)
    // for(int i=0; parray[i]; i++) 와 동일
    for(int i=0; parray[i] != NULL; i++)
        printf("parray[%d] = %s\n", i, parray[i]);

    return 0;
}
```

## 2. 명령행 인자 (Main 함수 인자) 개념

터미널에서 명령어와 함께 옵션이나 인수를 전달할 때 사용하며, `main()` 함수는 이를 `argc`와 `argv`로 전달받습니다.



* **사용 예시:**
    * `$ ls`: `int main()`
    * `$ ls for1.c`: `int main(int argc, char *argv[])`
    * `$ ls -l for1.c`: `int main(int argc, char **argv)` (`char *argv[]`와 `char **argv`는 동일함)

### 1) argc (Argument Count)
* **의미:** 인수(인자)의 개수 (정수형).
* **특징:** 프로그램 실행명까지 개수에 포함됩니다.
    * `$ ls` -> argc = 1
    * `$ ls for1.c` -> argc = 2
    * `$ ls -l for1.c` -> argc = 3

### 2) argv (Argument Value / Vector)
* **의미:** 각 인자 문자열의 시작 주소를 담고 있는 포인터 배열.
* **구조 (예: $ ls -l for1.c):**
    * `argv[0]`: "ls" 문자열의 주소
    * `argv[1]`: "-l" 문자열의 주소
    * `argv[2]`: "for1.c" 문자열의 주소
    * `argv[3]`: **NULL** (관례적으로 배열의 끝을 표시)

## 3. 실습: 명령행 인자 출력 (`arguments.c`)

```c
// arguments.c
#include <stdio.h>

int main(int argc, char *argv[])
{
    int i = 0;

    // argc 개수 출력
    printf("argc = %d\n", argc);

    // argv 포인터 배열 내의 문자열 출력
    for(; i < argc; i++)
        printf("argv[%d] = %s\n", i, argv[i]);

    return 0;
}
```

### 실행 테스트 결과
```bash
$ gcc -Wall -g -o arguments arguments.c

$ ./arguments
argc = 1
argv[0] = ./arguments

$ ./arguments 1 2 3
argc = 4
argv[0] = ./arguments
argv[1] = 1
argv[2] = 2
argv[3] = 3

$ ./arguments Hello C!
argc = 3
argv[0] = ./arguments
argv[1] = Hello
argv[2] = C!

$ ./arguments -l for1.c
argc = 3
argv[0] = ./arguments
argv[1] = -l
argv[2] = for1.c
```
