# 🔗 다중 포인터 (Triple Pointer 이상) 분석

포인터의 단계가 높아져도 본질은 동일합니다. 특정 변수의 주소를 저장하고, 역참조(`*`)를 통해 그 주소가 가리키는 실제 데이터에 접근하는 과정의 반복입니다.

---

## 1. 삼중 포인터를 이용한 문자열 배열 참조 (`ex_pointer5.c`)

이중 포인터 변수 `p`의 주소를 삼중 포인터 `pp`에 전달하여 문자열 배열을 제어하는 구조입니다.

### 소스 코드
```c
// ex_pointer5.c
#include <stdio.h>

void print_pointer3(char ***p);

int main()
{
    char *ch2[] = { "Linux", "Windows", "Unix", "Mac", NULL };
    char **p;

    // ch2의 시작 주소(&ch2[0])를 이중 포인터 p에 저장
    p = ch2;

    // 이중 포인터 변수 p의 주소를 삼중 포인터 함수에 전달
    print_pointer3(&p);

    return 0;
}

// 참조 구조 도식
// [  &p    ] --> [  &ch2[0]   ] --> [ "Linux" ... NULL ]
// pp : &p (char ***)
// *pp : p == ch2 (char **)
// (*pp)[0] == ch2[0] (char *) -> "Linux"

void print_pointer3(char ***pp)
{
    for(int i=0; (*pp)[i] != NULL; i++)
    {
        printf("%s\n", (*pp)[i]);       // 배열 인덱스 방식
        printf("%s\n", *(*pp + i));    // 포인터 연산 방식
    }
}
```

### 💻 실행 결과
```bash
$gcc -g -o ex_pointer5 ex_pointer5.c$ ./ex_pointer5
Linux
Linux
Windows
Windows
Unix
Unix
Mac
Mac
```

## 2. 다중 포인터의 연쇄 참조 (`ex_pointer6.c`)

포인터의 단계를 5단계까지 확장하여 최종적으로 `int`형 변수의 값에 접근하는 실습입니다.

### 소스 코드
```c
// ex_pointer6.c
#include <stdio.h>

int main()
{
    int number = 100;
    int *p = &number;
    int **pp = &p;
    int ***ppp = &pp;
    int ****pppp = &ppp;
    int *****ppppp = &pppp;

    printf("number     = %d\n", number);
    printf("*p         = %d\n", *p);
    printf("**pp       = %d\n", **pp);
    printf("***ppp     = %d\n", ***ppp);
    printf("****pppp   = %d\n", ****pppp);
    printf("*****ppppp = %d\n", *****ppppp);

    return 0;
}
```

### 📍 메모리 참조 구조 및 출력 결과
역참조 연산자(`*`)를 통해 주소를 타고 올라가면 결국 모두 동일한 `number`의 값에 도달합니다.



**참조 사슬:**
```text
*****ppppp      ****pppp       ***ppp        **pp         *p                number
[  &pppp  ] --> [  &ppp  ] --> [  &pp  ] --> [  &p  ] --> [ &number ]  --> [ 100 ]
```

**출력 결과:**
```text
*****ppppp      ****pppp       ***ppp        **pp         *p                number
   100            100            100           100          100               100
```

### 💻 실행 확인
```bash
$gcc -Wall -o ex_pointer6 ex_pointer6.c$ ./ex_pointer6
number     = 100
*p         = 100
**pp       = 100
***ppp     = 100
****pppp   = 100
*****ppppp = 100
```
