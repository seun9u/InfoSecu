# 🏗️ 구조체 변수의 초기화와 구조체 배열

구조체를 선언하면 그 안에는 쓰레기 값이 저장되어 있습니다. 이를 방지하기 위해 `memset()` 함수를 이용해 메모리를 0으로 초기화하거나, 선언과 동시에 초기화 리스트를 사용합니다.

---

## 1. memset() 함수를 이용한 메모리 초기화

`memset()`은 메모리의 내용을 특정 값으로 세팅할 때 사용합니다.

### 실습 1: 일반 배열 초기화 (`memset1.c`)
```c
#include <stdio.h>
#include <string.h>   // memset
// void *memset(void *s, int c, size_t n);
// 첫 번째 인수: 초기화할 변수의 주소(Call by reference)
// 두 번째 인수: 어떤 값으로 초기화 할 것인가
// 세 번째 인수: 초기화할 변수의 크기

int main()
{
    char buf[30] = "AAAAAAAAAAAAAA";

    // buf 변수를 0으로 초기화 한다.
    memset(buf, 0, sizeof(buf));

    return 0;
}
```

### 실습 2: 구조체 메모리 초기화 (`struct3.c`)
```c
#include <stdio.h>
#include <string.h>  // memset

struct member
{
    char name[30];  // 이름
    char phone[30]; // 전화번호
    int  age;       // 나이
};

int main()
{
    struct member m;

    printf("sizeof(m): %ld\n", sizeof(m));
    memset(&m, 'A', sizeof(m)); // 구조체 전체를 'A'로 채움

    return 0;
}
```

## 2. 구조체 배열 (Array of Structures)

구조체 변수를 여러 개 묶어서 관리할 때 사용하며, 관리상의 큰 이점이 있습니다.

* **선언 형식:** `struct 구조체명 배열명[개수];`
* **접근 방법:** `배열명[인덱스].멤버변수명;`

### 실습 3: 학생 성적 관리 프로그램 (`struct4.c`)

```c
#include <stdio.h>
#define COUNT 3

struct jumsu
{
    char name[30];   // 이름
    int kor;         // 국어 점수
    int eng;         // 영어 점수
    int math;        // 수학 점수
    int total;       // 총점
    double average;  // 평균
};

int main()
{
    struct jumsu student[COUNT];  // 3명의 학생 구조체 배열
    int i;

    // 3. 학생 점수 입력
    for(i = 0; i < COUNT; i++)
    {
        printf("===== %d 번째 학생 =====\n", i + 1);
        printf("이름 : "); scanf("%s", student[i].name);
        printf("국어 점수 : "); scanf("%d", &student[i].kor);
        printf("영어 점수 : "); scanf("%d", &student[i].eng);
        printf("수학 점수 : "); scanf("%d", &student[i].math);

        student[i].total = student[i].kor + student[i].eng + student[i].math;
        student[i].average = student[i].total / 3.0;
    }

    putchar('\n');

    // 4. 학생 점수 출력
    for(i = 0; i < COUNT; i++)
    {
        printf("===== %d 번째 학생 %s's 점수 =====\n", i + 1, student[i].name);
        printf("총점 : %d, 평균 : %.2f\n", student[i].total, student[i].average);
    }

    return 0;
}
```

## 3. 구조체 변수 초기화 리스트 (`struct5.c`)

선언과 동시에 중괄호 `{}`를 사용하여 멤버 순서대로 초기화할 수 있습니다.

```c
#include <stdio.h>

struct jumsu
{
    char name[30];
    int kor, eng, math, total;
    double average;
};

int main()
{
    struct jumsu myjumsu = { "MrHong", 100, 100, 100, 300, 100 };

    printf("이름 : %s, 총점 : %d, 평균 : %.2f\n", 
            myjumsu.name, myjumsu.total, myjumsu.average);

    return 0;
}
```
