## 2. Ubuntu 24.04-3-desktop 세팅하기

### 1. 패키지 리스트 업데이트
```bash
sudo apt update
```

### 2. 전체 화면 설정
전체화면이 안 되는 문제 -> `open-vm-tools` + `open-vm-tools-desktop` 설치

```bash
sudo apt install -y open-vm-tools open-vm-tools-desktop
```
* 설치 후 상단 **full screen mode** 클릭 시 전체화면 적용

### 3. 소프트웨어 업데이트
* 팝업 창 뜨면 업데이트 진행

## 3. gcc 설치 및 컴파일러 디버거 설치

gcc는 C언어의 컴파일러가 들어있으므로 이를 설치해야 한다.

```bash
sudo apt install -y build-essential
```
* `build-essential`: gcc뿐만 아니라 g++/make/표준 헤더 등 개발에 필요한 기본 묶음이 같이 설치된다.

### 설치 확인 명령어
```bash
gcc --version
g++ --version
make --version
gdb --version
```

### 기초 명령어
* `pwd` : 현재 작업 디렉토리(위치) 확인
* `cd` : 디렉토리 이동
* `mkdir` : 디렉토리(폴더) 생성
* `ls` : 파일 목록 확인
* `cat [파일이름]` : 파일 미리 보기
> **Shift + Insert = Ubuntu 붙여넣기**

## 4. Ubuntu 내 VS Code 설치

```bash
sudo apt update
sudo apt install -y wget gpg
```

VS Code를 설치할 때 Snap도 가능하지만 실무/호환 측면에서 **공식 repo (패키지 저장소) 설치를 추천**한다.
VS Code를 Microsoft 공식 APT 저장소로 등록해서 `apt install code` / `apt upgrade`로 업데이트까지 자동으로 받게 만드는 것이고, 이때 저장소 서명키(공개키)를 키링에 저장한다.
그 키로만 검증되는 저장소 항목을 `sources.list.d`에 추가하는 절차이다.

### 설치 명령어
```bash
wget -qO- [https://packages.microsoft.com/keys/microsoft.asc](https://packages.microsoft.com/keys/microsoft.asc) \
 | gpg --dearmor \
 | sudo tee /usr/share/keyrings/microsoft.gpg > /dev/null

echo "deb [arch=amd64 signed-by=/usr/share/keyrings/microsoft.gpg] [https://packages.microsoft.com/repos/code](https://packages.microsoft.com/repos/code) stable main" \
 | sudo tee /etc/apt/sources.list.d/vscode.list

sudo apt update
sudo apt install -y code
```

## 5. VS Code 실행 후 세팅

### 1. MS사 확장팩 설치
* 한글 확장팩 설치 (`kor` 검색)
* C/C++ 확장팩 설치 (`C/C++` 검색)

### 2. 작업 폴더 선택
* 바탕화면에 만들어둔 `[c_workspace]` 폴더 선택 후 우측 상단 **[열기]**
* 파일의 작성자 신뢰 팝업 생성 시, **[모든 파일의 작성자 신뢰]** 체크 표시 후 **[파란색 신뢰]** 클릭

### 3. 소스 오픈
* 기존에 만들어둔 `hello.c` 오픈 (없을 시 `.c` 만들기)
* 오른쪽 하단에 확장팩 설치 팝업이 뜨면, **[설치]** 클릭

### 4. 폰트 크기 조절
* 기본 설정(`Ctrl` + `,`) 창 진입 후, `zoom` 검색
* **[Editor: Mouse Wheel zoom]** 체크

### 5. 컴파일 및 실행
* 세팅 완료 시, `Ctrl` + `F5`를 눌러 컴파일 진행
* 가운데 상단 화면에 **C/C++: gcc 활성 파일 빌드 및 디버그** 나오면 선택
* 하단 터미널에 컴파일된 실행 결과가 출력됨
