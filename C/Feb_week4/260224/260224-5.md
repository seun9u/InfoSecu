## 실습.[LAB] 함수에서 포인터의 리턴 사용하기

[ Text, Code | Data | BSS | Heap | Stack ]

### 1. 변수의 종류 및 메모리 저장 영역

* **지역 변수 (Local Variable)**
    - 함수 안에서 생성하며 해당 함수에서만 사용 가능.
    - 선언 시 `auto` 키워드를 사용하지만 생략 가능하다.
    - **Stack** 메모리에 저장된다.
    - 함수가 종료되면 사용하고 있는 Stack 공간이 사라지므로 다른 함수에서 접근할 수 없다.
* **정적 변수 (Static Variable)**
    - 함수 안에서 생성했지만 프로그램 종료 시까지 유지된다.
    - 선언 시 `static` 키워드를 사용한다.
    - **Data | BSS** 영역에 저장된다.
        - **Data**: 초기화된 정적 변수
        - **BSS**: 초기화되지 않은 정적 변수
* **전역 변수 (Global Variable)**
    - 함수 밖에서 생성하며 한 파일 혹은 다른 파일의 함수에서도 접근 가능하다.
    - 다른 파일에서 접근할 때는 `extern` 키워드를 사용한다.
    - **Data | BSS** 영역에 저장된다.
        - **Data**: 초기화된 전역 변수
        - **BSS**: 초기화되지 않은 전역 변수

### 2. 함수 리턴 테스트 (포인터 반환 시 주의사항)

#### [실습 1] 지역 변수 주소 리턴 (실패: `ex_pointer7.c`)

```c
#include <stdio.h>
char *function();

int main(){
    char *retp;
    retp = function();
    printf("%s\n", retp);
    return 0;
}

char *function(){
    char name[] = "James Hong";  // 지역 변수 (Stack)
    char *p;
    p = name;
    /*
       p              name
       +-----+        +--------------+
       | 100 |------->| James Hong\0 |
       +-----+        +--------------+
                      100
    */
    // 지역변수의 주소는 함수가 끝나면서 사라진다. 
    // 그러므로 지역변수의 주소를 리턴할 수 없다.
    return p;  
}
```
* **실행 결과:** (출력이 안 됨)

#### [실습 2] 정적 변수 주소 리턴 (성공: `ex_pointer8.c`)
```c
#include <stdio.h>
char *function();

int main(){
    char *retp;
    retp = function();
    printf("%s\n", retp);
    return 0;
}

char *function(){

```
* **실행 결과:** `James Hong`

### 3. 동적 메모리 할당 (Heap 영역)

#### [실습 4] malloc을 이용한 정수형 할당 (`malloc1.c`)
```c
#include <stdio.h>
#include <stdlib.h>

int main(){
    // Heap 에 4byte 공간이 할당되고 생성된 주소를 리턴한다.
    int *a = (int *)malloc(sizeof(int));  // 동적 메모리 생성
    *a = 1;

    printf("*a = %d\n", *a);  // 1
    // &a: 스택 메모리 주소, a: 할당받은 힙 메모리 주소
    printf("&a = %p, a = %p\n", &a, a);

    free(a);  // 동적 메모리 해제 필수
    return 0;
}
```
* **메모리 레이아웃:**
    `Low memory <--- [Text | Data | BSS | Heap | Stack] ---> High memory`

### 4. 구조체 동적 할당 및 포인터 리턴 실습 (`struct9.c`)



```c
#include <stdio.h>
#include <stdlib.h>  // malloc
#include <string.h>  // strncpy

struct member{
    char name[20];
    int age;
};

struct member *create_member(const char *name, int age);

int main(){
    struct member *m;
    m = create_member("Hgd", 20);

    printf("name = %s, age = %d\n", m->name, m->age );
    
    free(m); // 사용 완료 후 동적 메모리 해제
    return 0;
}

struct member *create_member(const char *name, int age){
    // 동적 메모리를 생성한다. (Heap 영역)
    // void *malloc(size_t size)은 모든 자료형의 주소를 리턴하는 전천후 포인터.
    struct member *p = (struct member*) malloc(sizeof(struct member));
    if(p == NULL)  // 할당 실패 시 NULL 리턴
        return NULL;

    // 안전한 복사를 위해 strncpy 사용 및 널 문자로 끝처리
    strncpy(p->name, name, sizeof(p->name)-1);
    p->name[sizeof(p->name)-1] = '\0';
    p->age = age;

    return p; // 생성된 구조체의 Heap 주소 반환
}
```
* **실행 결과:** `name = Hgd, age = 20`
