## 실습.[LAB] 다중 if문과 switch-case문 분석

### 1. 다중 if문 (if-else if-else)
여러 조건 중 하나가 참이면 해당 블록을 실행하고 나머지는 건너뜁니다.

#### [소스 코드] ifelseif.c

```c
#include <stdio.h>

int main()
{
    int input_number;

    fprintf(stdout, "input number: ");
    scanf("%d", &input_number);

    if(input_number == 1) {
        printf("입력값은 1입니다.\n");
    } else if(input_number == 2) {
        printf("입력값은 2입니다.\n");
    } else if(input_number == 3) {
        printf("입력값은 3입니다.\n");
    } else if(input_number == 4) {
        printf("입력값은 4입니다.\n");
    } else if(input_number == 5) {
        printf("입력값은 5입니다.\n");
    } else {
        printf("입력한 값은 1 ~ 5가 아닙니다.\n"); 
    }

    return 0;
}
```

#### 실행 결과
```bash
$ gcc -o ifelseif ifelseif.c

$ ./ifelseif
input number: 7
입력한 값은 1 ~ 5가 아닙니다.

$ ./ifelseif
input number: 1
입력값은 1입니다.

$ ./ifelseif
input number: 4
입력값은 4입니다.
```

### 2. switch-case문
정해진 상숫값에 따라 분기합니다. `break`가 없으면 다음 `case`로 계속 흘러가는 **Fall-through** 특징이 있습니다.

#### [소스 코드] swichcase2.c

```c
#include <stdio.h>

int main()
{
    int input_number;

    fprintf(stdout, "번호 입력(1 ~ 5): ");
    scanf("%d", &input_number);

    switch(input_number)
    {
        case 1:
            // printf("...\n"); fprintf(stdout, "...\n");
            puts("입력한 값은 1입니다.");
        case 2:
            puts("입력한 값은 2입니다.");
        case 3:
            puts("입력한 값은 3입니다.");
            break;
        case 4: puts("입력한 값은 4입니다.");  break;
        case 5: puts("입력한 값은 5입니다.");  break;
        default:  // 1 ~ 5가 아닌 경우
            puts("1 ~ 5까지만 입력해야 합니다.");
            break;
    }

    return 0;
}
```

#### 실행 결과 분석
```bash
$ gcc -o swichcase2 swichcase2.c

# 1 입력 시: break를 만날 때까지 1, 2, 3이 모두 출력됨
$ ./swichcase2
번호 입력(1 ~ 5): 1
입력한 값은 1입니다.
입력한 값은 2입니다.
입력한 값은 3입니다.

# 2 입력 시: case 2부터 시작하여 2, 3이 출력됨
$ ./swichcase2
번호 입력(1 ~ 5): 2
입력한 값은 2입니다.
입력한 값은 3입니다.

# 3 입력 시: 해당 case만 출력 후 break
$ ./swichcase2
번호 입력(1 ~ 5): 3
입력한 값은 3입니다.

# 범위 밖의 값 입력 시: default 실행
$ ./swichcase2
번호 입력(1 ~ 5): 6
1 ~ 5까지만 입력해야 합니다.
```
