## 실습.[LAB] 구조체를 활용한 사용자 검색 프로그램

### /etc/passwd 파일 구조
* 사용자의 정보가 저장된 파일입니다.
* 구분자 **`:`(콜론)**으로 데이터가 나뉩니다.
* 예: `root:x:0:0:root:/root:/bin/bash` (1 2 3 4 5 6 7 순서)
    1. **Username**: 사용자 계정명
    2. **Password**: 비밀번호 표시 (x)
    3. **User ID**: 사용자 ID
    4. **Group ID**: 그룹 ID
    5. **Real name**: 실제 이름
    6. **Home directory**: 홈 디렉토리
    7. **Shell program**: 기본 쉘 프로그램

---

### [실습 코드] linuxUserInfo.c

```c
#include <sys/types.h>
#include <pwd.h>
#include <stdio.h>

/*
 * /etc/passwd 구조체 정의
 * struct passwd
 * {
 * char *pw_name;           // Username.
 * char *pw_passwd;         // Password.
 * __uid_t pw_uid;          // User ID.
 * __gid_t pw_gid;          // Group ID.
 * char *pw_gecos;          // Real name.
 * char *pw_dir;            // Home directory.
 * char *pw_shell;          // Shell program.
 * };
 */

int main(int argc, char *argv[])
{
    if(argc == 1)
    {
        fprintf(stderr, "사용법: %s username\n", argv[0]);
        return 1;  // 종료 코드 1
    }

    // 사용자를 /etc/passwd에서 검색한다.
    struct passwd *pw = getpwnam(argv[1]);

    if(pw == NULL) { // 검색에 실패하면 프로세스를 종료한다.
        fprintf(stderr, "%s 사용자가 존재하지 않습니다.\n", argv[1]);
        return 2; // 종료 코드 2
    }

    // 검색에 성공하면 사용자 정보를 출력한다.
    printf(">>> 검색된 사용자 정보 <<<\n"
           "Username: %s\n"
           "Password: %s\n"
           "User ID: %d\n"
           "Group ID: %d\n"
           "Real name: %s\n"
           "Home directory: %s\n"
           "Shell program: %s\n",
           pw->pw_name, pw->pw_passwd, pw->pw_uid,
           pw->pw_gid,  pw->pw_gecos,  pw->pw_dir, pw->pw_shell);

    return 0;
}
```

### 실행 결과 및 테스트

## 실습.[LAB] 리눅스/윈도우에서 조건부 컴파일하기

### 전처리기(Preprocessor) 조건부 컴파일 지시문
* **컴파일 단계:** 소스 코드(.c) > 전처리 > 어셈블리 > 오브젝트 > 실행 파일
* **#ifdef / #else / #endif:** 매크로 정의 여부에 따라 컴파일 포함 여부를 결정합니다.

### 플랫폼별 주요 매크로
* **Windows:** `_WIN32` (공통), `_WIN64` (64비트 전용)
* **Linux:** `__linux__`, `__unix__`
* **Mac:** `__APPLE__`, `__MACH__`

### [실습 코드] platform_ifdef.c

```c
#include <stdio.h>

int main()
{
#if defined(_WIN32)  // Windows 이면
    #if defined(_WIN64)  // Windows 64-bit 전용
        printf("64bit MS Windows\n");
    #else  // Windows 32-bit 전용
        printf("32bit MS Windows\n");
    #endif
#elif defined(__APPLE__)
    #if TARGET_OS_MAC && !TARGET_OS_IPHONE
        // macOS
    #elif TARGET_OS_IPHONE
        // iOS (iPhone/iPad)
    #else
        // 기타 Apple (watchOS/tvOS 등)
    #endif
#else
    printf("Linux\n");
#endif

    printf("조건부 컴파일 지시문 이후에 실행\n");

    return 0;
}
```

### 실행 결과 (Linux 환경)
```bash
$gcc -o platform_ifdef platform_ifdef.c$ ./platform_ifdef
Linux
조건부 컴파일 지시문 이후에 실행
```
